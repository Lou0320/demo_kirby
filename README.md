# Kirby Party — Démo créative

## Présentation

Ce projet est une démo d'environ 3 minutes montrant un petit Kirby qui nous invite à faire la fête avec lui et un ami. C'est une réalisation créative produite dans le cadre du concours API8 et également pour le cours d'image synthèse et son (L2) donné par le professeur Farès Belhadj.

## Contexte

- Concours : API8 (édition 2025) — le projet a séduit le jury et a remporté le premier prix de la Track 1.
- Cours : Image synthèse et son — Licence 2, professeur Farès Belhadj.

# Kirby Party — Creative demo (English translation)

## Presentation

This project is an approximately 3-minute demo featuring a small Kirby who invites us to party with him and a friend. It was created for the API8 competition and also for the "Image, Synthesis and Sound" course (L2) taught by Professor Farès Belhadj.

## Context

- Competition: API8 (2025 edition) — the project won the jury's favor and received the first prize in Track 1.
- Course: Image, Synthesis and Sound — Bachelor Year 2, Professor Farès Belhadj.

## Technologies

The project is implemented using GL4Dummies and SDL for windowing and audio.

## Content

The demo presents a short animated scene with soundtrack featuring a Kirby-like character and a friend in a festive setting. Approximate duration: 3 minutes.

## Author

Lou-Ann Coquard-Morel
